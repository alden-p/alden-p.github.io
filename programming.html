Orpheus Project

This is a personal project of mine which automatically writes original pieces of music using machine learning methods. The project is functional and requires python 3 and windows system to run. In future iterations the project will utilize midi to improve sound quality, have improved UI and see some tweaks to the machine learning. The project is also open source so feel free to contribute, check it out on my github here.
