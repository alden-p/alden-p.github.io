Orpheus Project

This is a personal project of mine which is intended to write original pieces of music using machine learning methods. The project is functional but still in its early stages, and currently it relies on windows system sounds to produce musical effects. In future iterations the project will likly utilize midi to improve sound quality. Better UI and tweaks to the machine learning can also be expected in the future. The project is also open source so feel free to contribute, check it out on my github here.
